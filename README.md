# 🎓 Student Profile Management System

## 📘 Project Overview
This project is a simple Student Profile Management System developed for the Fundamentals of Data Science module as a Final Project.

The system is designed to manage student information such as:
- Personal details
- Academic grades
- Extracurricular activities (ECA)

It uses text files as a database and provides different features for admin and student users.

---

## 🎯 Features

### 🔐 Login System
- Secure login using username and password
- Role-based access (Admin / Student)

---

### 👨‍💼 Admin Features
- Add new users
- View all users
- Delete users
- Add student grades
- Add extracurricular activities
- Calculate average marks
- View performance analytics:
  - Grade trends (graph)
  - ECA impact
  - Performance alerts

---

### 🎓 Student Features
- View profile
- View grades
- View ECA activities
- Update profile

---

## 📁 Project Files
main.py → Main program file
users.txt → Stores user information
passwords.txt → Stores login credentials
grades.txt → Stores student marks
eca.txt → Stores extracurricular activities


---

## ⚙️ Technologies Used
- Python
- Text Files (for data storage)
- Matplotlib (for graphs)
- Colorama (for UI styling)

---

## ▶️ How to Run the Project

1. Make sure Python is installed  
2. Install required libraries:
   
   pip install matplotlib colorama
   
3. Run the program:
    

python main.py
